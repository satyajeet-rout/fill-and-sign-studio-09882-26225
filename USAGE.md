### 2. Form Field Editing

#### Detected Fields
The app automatically detects form fields in your PDF:

#### Editing Fields
- **In-Document**: Click directly on form fields in the PDF to edit
- **Sidebar Panel**: Use the sidebar to view and edit all fields at once
- **Real-time Sync**: Changes in sidebar sync with the PDF view instantly

### 3. Signature Management

#### Adding Signatures
1. Click "Add Signature" in the sidebar
2. Upload an image file (PNG, JPG, or SVG)
3. Your signature will appear on the current page

#### Positioning Signatures
- **Drag**: Click and drag to move the signature
- **Resize**: Use the corner handle to resize
- **Delete**: Click the trash icon to remove

#### Best Practices
- Use transparent PNG for best results
- Recommended size: 400x200px
- Keep file size under 1MB

## üí° Tips & Tricks

### Efficiency
- Fill common fields in sidebar first
- Use Tab key to move between fields
- Zoom in for precise signature placement
- Review all pages before saving

### Quality
- Use high-resolution signature images
- Check alignment before saving
- Preview all pages after editing
- Save a copy of the original PDF

### Troubleshooting

#### PDF Not Loading
- Ensure file is a valid PDF
- Check file isn't corrupted
- Try re-opening the file
- Maximum recommended size: 50MB

#### Form Fields Not Detected
- Some PDFs don't have embedded form fields
- Fields will need to be manually overlaid
- Consider using a form-fillable template

#### Signature Placement Issues
- Zoom in for precise placement
- Use grid alignment mentally
- Test print if needed
- Adjust size before positioning
